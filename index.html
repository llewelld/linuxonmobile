Linux on Mobile

Welcome to the Linux on Mobile project page.

Nemomobile developer Jozef Mlich spoke about Linux on Mobile at installfest.cz where he discussed "The Last Mile Between Users and Linux Phones". The live stream recording is available so you can still play Linux on Mobile bingo!

Linux on Mobile is a collaboration between multiple different projects. This page links to all of the projects so you can find out all the information you need to get to know them.
